
# 🔌 Browser Extensions Manager UI

Este projeto é parte da atividade **Lista 05** da disciplina de Desenvolvimento Front-end. O objetivo foi construir uma interface de gerenciamento de extensões de navegador com base em um protótipo fornecido.

## 🎯 Objetivo

Criar uma aplicação responsiva que:

- Exiba uma lista de extensões com nome, ícone e descrição
- Permita ativar/desativar cada extensão individualmente
- Remova extensões da lista
- Filtre as extensões por status (Todas, Ativas, Inativas)
- Implemente tema claro e escuro com troca de ícone (sol/lua)
- Seja responsiva e acessível para diferentes tamanhos de tela

## 🧪 Tecnologias Utilizadas

- HTML5
- CSS3 (Custom Properties para tema)
- JavaScript (DOM, eventos, fetch de JSON)
- Design com base nos arquivos fornecidos em JPG

## 📁 Estrutura de Arquivos

📦 projeto  
├── index.html  
├── style.css  
├── script.js  
├── data.json  
└── assets/images/ (ícones e logos)

## ⚙️ Como executar o projeto

1. Clone ou baixe o repositório  
2. Abra o arquivo `index.html` em um navegador ou use Live Server  
3. A interface será carregada com os dados do `data.json`

## 🌓 Tema Claro/Escuro

- O botão no topo da página alterna entre os temas  
- O ícone muda automaticamente (☀️ para 🌙 e vice-versa)
